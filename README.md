# draftworks

My test bench for building things — professional projects, experiments, and side work.

## What's Here

Each top-level directory is an independent project. Projects range from quick experiments to longer-running work, across any language or domain.

## Adding a Project

1. Create a new directory at the root with a descriptive name.
2. Add a `README.md` inside it explaining what the project is and how to run it.
3. Keep dependencies self-contained within the project directory.
